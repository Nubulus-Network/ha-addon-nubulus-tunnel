# Nubulus Network Add-ons for Home Assistant

This repository contains unofficial Home Assistant add-ons maintained by
[Nubulus Network](https://nubulusnetwork.es).

## Adding this repository to Home Assistant

1. Go to **Settings → Add-ons → Add-on Store** in your Home Assistant instance.
2. Click the overflow menu (⋮) in the top-right corner and select **Repositories**.
3. Paste the repository URL and click **Add**:
   ```
   https://github.com/Nubulus-Network/ha-addon-nubulus-tunnel
   ```

## Available add-ons

### [Nubulus Tunnel](nubulus-tunnel/)

Expose your Home Assistant to the internet via Nubulus Network tunnels — no
open ports, no public IP address, and no router configuration required.

See the [add-on documentation](nubulus-tunnel/DOCS.md) for installation and
configuration instructions.
